# GudangIn - Sistem Informasi Manajemen Inventaris Gudang Sembako
## Fitur Aplikasi:
- Manajemen Data Master (Tambah, Ubah, Hapus) menggunakan Array of Struct
- Pencatatan Transaksi Masuk & Keluar dengan satuan Kilogram (kg)
- Pencarian berbasis Sequential Search dan Binary Search
- Pengurutan Stok berbasis Selection Sort dan Insertion Sort
- Statistik total nilai aset dan sisa stok menipis
